# Shopper Spectrum

## Customer Segmentation and Product Recommendation in E-Commerce

### Project Description
This project analyzes online retail customer transactions using Machine Learning.

### Features
- Customer Segmentation using RFM Analysis
- K-Means Clustering
- Product Recommendation System
- Streamlit Web Application

### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn
